# Mobile App Guide

Install and use the Aura mobile app.

## Installation

### Android

:::warning App Not Available on Play Store Yet
The Aura mobile app is **not yet available** on the Google Play Store.
:::

**To install the app:**

1. Visit your school's Aura web portal
2. Navigate to the **Downloads** or **Mobile App** section
3. Download the APK file directly from the Aura page
4. Enable "Install from Unknown Sources" in your Android settings:
   - Go to **Settings** → **Security** → **Unknown Sources**
   - Or **Settings** → **Apps** → **Special Access** → **Install Unknown Apps**
5. Open the downloaded APK file
6. Follow the installation prompts
7. Open the app and log in

:::tip
Contact your school administrator for the direct download link to the Aura mobile app.
:::

### iOS (Coming Soon)
iOS version is under development.

## Features

- Quick check-in with face scan
- View attendance records
- Receive push notifications
- Access event schedules
- Geolocation-based check-in

## First Time Setup

1. Open the app
2. Enter your school code
3. Log in with your credentials
4. Allow camera and location permissions
5. Complete face enrollment (optional)

## Using the App

### Check In
1. Tap "Check In"
2. Use face scan at kiosk or mobile camera
3. Ensure you're at the event location
4. Wait for confirmation

### View Attendance
1. Tap "Attendance"
2. See your records
3. Filter by date

## Troubleshooting

**App won't open?**
- Update to latest version
- Clear app cache
- Reinstall if needed

**Can't check in?**
- Ensure you're at the event location
- Check if event is active
- Verify GPS/location is enabled
- Try face scan again with better lighting
- Ask operator for manual entry
