# Troubleshooting Guide

Common issues and solutions for Aura users.

## Face Recognition Issues

### Face Scan Not Working

**Problem**: Face recognition fails or doesn't detect face

**Solutions**:
1. **Improve Lighting**: Ensure adequate, even lighting on your face
2. **Remove Obstructions**: Take off glasses, hats, or masks
3. **Position Correctly**: Face the camera directly, 30-50cm away
4. **Hold Still**: Keep your head steady for 2-3 seconds
5. **Clean Camera**: Wipe camera lens if dirty

### Anti-Spoofing Detection

**Problem**: System rejects face scan saying "Spoofing detected"

**Solutions**:
- Ensure you're using live camera, not a photo/video
- Move slightly to show you're a real person
- Check lighting isn't creating unusual shadows
- Try again with better camera angle

### Face Not Enrolled

**Problem**: "Face not enrolled" error message

**Solutions**:
1. Visit enrollment kiosk at your school
2. Complete face enrollment process
3. Wait 5-10 minutes for system to process
4. Try checking in again

## Geolocation Issues

### Outside Geofence

**Problem**: "You are outside the event geofence" error

**Solutions**:
1. **Move Closer**: Walk closer to the event venue
2. **Enable GPS**: Ensure location services are enabled
3. **Wait for Signal**: GPS may need time to acquire accurate location
4. **Check Accuracy**: Location accuracy should be < 50 meters

### Location Permission Denied

**Problem**: App can't access your location

**Solutions**:

**On Android**:
1. Settings → Apps → Aura → Permissions
2. Enable "Location" permission
3. Select "Allow all the time" or "Allow only while using"

**On iOS**:
1. Settings → Privacy → Location Services
2. Find Aura app
3. Select "While Using the App"

### GPS Not Accurate

**Problem**: GPS shows wrong location

**Solutions**:
- Go outdoors for better GPS signal
- Restart your device
- Toggle Airplane mode on/off
- Wait 30-60 seconds for GPS to stabilize

## Check-In Issues

### Already Checked In

**Problem**: "You are already checked in to this event"

**Explanation**: You can only check in once per event

**Solution**: If this is an error, contact event organizer

### Event Not Started

**Problem**: "Event has not started yet"

**Solution**: Wait until the event start time, then try again

### Event Already Ended

**Problem**: "Event has already ended"

**Solution**: Contact event organizer if you need late check-in

### Check-In Window Closed

**Problem**: "Check-in window has closed"

**Explanation**: Events have a limited check-in period

**Solution**: Arrive earlier next time, or contact organizer for excused absence

## Mobile App Issues

### App Won't Load

**Solutions**:
1. Check internet connection (WiFi or mobile data)
2. Clear app cache
3. Update to latest version
4. Restart device
5. Reinstall app if problem persists

### App Crashes

**Solutions**:
1. Update to latest version
2. Clear app cache and data
3. Restart device
4. Check available storage space (need at least 100MB free)
5. Report crash to IT support with error details

### Slow Performance

**Solutions**:
- Close other apps running in background
- Clear app cache
- Check internet speed
- Restart device
- Update app to latest version

### Can't Login

**Problem**: Login fails with incorrect credentials

**Solutions**:
1. **Check Credentials**: Verify username and password
2. **Reset Password**: Contact IT support or campus admin
3. **Account Locked**: May be locked after multiple failed attempts
4. **Network Issue**: Check internet connection

## Notification Issues

### Not Receiving Notifications

**Solutions**:

**Check App Permissions**:
1. Settings → Apps → Aura → Notifications
2. Enable "Allow notifications"
3. Enable specific notification types

**Check Do Not Disturb**:
- Disable Do Not Disturb mode
- Or add Aura to exceptions

**Re-enable in App**:
1. Open Aura app
2. Go to Settings → Notifications
3. Enable desired notification types

### Too Many Notifications

**Solution**:
1. Open Aura app
2. Settings → Notifications
3. Disable unwanted notification types
4. Adjust notification frequency

## Attendance Record Issues

### Attendance Not Showing

**Problem**: Check-in successful but not in attendance history

**Solutions**:
1. **Wait**: Records may take 1-2 minutes to sync
2. **Refresh**: Pull down to refresh attendance list
3. **Check Date Filter**: Ensure correct date range selected
4. **Contact Support**: If still missing after 10 minutes

### Wrong Attendance Status

**Problem**: Marked absent but you checked in

**Solutions**:
1. Check if you checked in during valid time window
2. Verify you were within geofence
3. Contact event organizer with proof (screenshot of check-in confirmation)
4. Request manual correction

## Profile Issues

### Can't Update Profile

**Problem**: Profile changes don't save

**Solutions**:
- Check internet connection
- Ensure all required fields are filled
- Try again later if server is busy
- Contact IT support if problem persists

### Profile Picture Won't Upload

**Solutions**:
- Check file size (max 5MB)
- Use supported formats (JPG, PNG)
- Compress image if too large
- Check internet connection

## Event Issues

### Can't See Event

**Problem**: Event not showing in your list

**Possible Reasons**:
- Event is for different section/program
- Event hasn't been published yet
- You're not in the target audience
- Event was cancelled

**Solution**: Contact event organizer

### Event Details Wrong

**Problem**: Event time, location, or details are incorrect

**Solution**: Contact event organizer to update details

## System Errors

### "Server Error" Message

**Solutions**:
1. Wait a few minutes and try again
2. Check if system is under maintenance
3. Report to IT support if error persists

### "Network Error" Message

**Solutions**:
- Check internet connection
- Switch between WiFi and mobile data
- Restart app
- Try again in a few minutes

### "Session Expired" Message

**Solution**: Simply log in again

- Screenshot of error (if any)
- Steps to reproduce the issue

- Attendance corrections
- Late check-in requests
- Event-specific issues

**When to Contact**:
- Account permissions
- Role assignment issues
- School-wide concerns
- Policy questions
