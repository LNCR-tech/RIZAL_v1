# Events

Browse and manage school events.

## Viewing Events

1. Go to Events page
2. See upcoming events
3. Click on an event for details

## Event Details

Each event shows:
- Event name and description
- Date and time
- Location
- Check-in requirements
- Attendance status

## For Student Government: Creating Events

SSG, SG, and ORG officers can create events:

1. Click "Create Event"
2. Fill in event details
3. Set attendance window
4. Configure check-in methods:
   - Face recognition (primary)
   - Manual entry (backup)
   - Geolocation verification (optional)
5. Set geofence radius if using location
6. Save and publish
