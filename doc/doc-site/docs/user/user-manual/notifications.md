# Notifications

Stay informed about events, attendance, and system updates through Aura's notification system.

## Notification Types

### 📅 Event Notifications

**Event Reminders**
- Sent 24 hours before event
- Sent 1 hour before event
- Includes event details and location

**Event Updates**
- Event time or location changes
- Event cancellations
- New events published

### ✅ Attendance Notifications

**Check-In Confirmation**
- Immediate confirmation after successful check-in
- Shows event name and timestamp

**Attendance Warnings**
- Low attendance alerts
- Approaching sanction thresholds
- Missed required events

### 🔔 System Notifications

**Account Updates**
- Password changes
- Profile updates
- Role changes

**System Announcements**
- Maintenance schedules
- New features
- Important updates

## Managing Notifications

### Enable/Disable Notifications

#### In Mobile App

1. Open Aura app
2. Tap **Profile** → **Settings**
3. Tap **Notifications**
4. Toggle notification types on/off:
   - Event Reminders
   - Attendance Alerts
   - System Announcements

#### In Device Settings

**Android**:
1. Settings → Apps → Aura
2. Tap **Notifications**
3. Enable/disable notification categories
4. Adjust importance level

**iOS**:
1. Settings → Notifications → Aura
2. Enable **Allow Notifications**
3. Choose alert style
4. Enable/disable sounds and badges

### Notification Preferences

**Frequency Settings**:
- **All**: Receive all notifications
- **Important Only**: Critical alerts only
- **None**: Disable all notifications (not recommended)

**Quiet Hours**:
- Set time range to mute notifications
- Example: 10 PM - 7 AM
- Emergency alerts still come through

**Delivery Method**:
- Push notifications (mobile)
- In-app notifications
- Email notifications (if configured)

## Notification Center

### Accessing Notification Center

1. Open Aura app
2. Tap **Bell icon** in top right
3. View all recent notifications

### Notification Actions

**Mark as Read**:
- Tap notification to mark as read
- Or swipe right and tap "Mark Read"

**Delete Notification**:
- Swipe left on notification
- Tap "Delete"

**Clear All**:
- Tap "Clear All" button
- Removes all read notifications

### Notification Details

Tap any notification to view:
- Full message content
- Related event or attendance record
- Action buttons (if applicable)

## Troubleshooting

### Not Receiving Notifications

**Check App Permissions**:
1. Ensure notifications are enabled in app settings
2. Check device notification permissions
3. Verify internet connection

**Check Do Not Disturb**:
- Disable Do Not Disturb mode
- Or add Aura to DND exceptions

**Re-register for Notifications**:
1. Disable notifications in app
2. Close app completely
3. Re-enable notifications
4. Restart app

### Delayed Notifications

**Possible Causes**:
- Poor internet connection
- Device battery saver mode
- Background app restrictions

**Solutions**:
- Check internet connection
- Disable battery optimization for Aura
- Allow background data usage

### Too Many Notifications

**Adjust Frequency**:
1. Settings → Notifications
2. Select "Important Only"
3. Disable non-essential categories

**Set Quiet Hours**:
1. Settings → Notifications → Quiet Hours
2. Set start and end time
3. Save preferences

### Notification Sound Issues

**Change Notification Sound**:

**Android**:
1. Settings → Apps → Aura → Notifications
2. Tap notification category
3. Select "Sound"
4. Choose preferred sound

**iOS**:
1. Settings → Notifications → Aura
2. Tap "Sounds"
3. Select notification sound

## Best Practices

### For Students

✅ **Enable Event Reminders**: Never miss important events
✅ **Keep Notifications On**: Stay informed about attendance
✅ **Check Regularly**: Review notification center daily
✅ **Act Promptly**: Respond to attendance warnings quickly

### For Event Managers

✅ **Send Timely Updates**: Notify attendees of changes ASAP
✅ **Use Clear Titles**: Make notifications easy to understand
✅ **Avoid Spam**: Only send necessary notifications
✅ **Test Notifications**: Verify delivery before events

## Privacy & Data

### What We Track

- Notification delivery status
- Read/unread status
- User preferences

### What We Don't Track

- Notification content after deletion
- Personal device information
- Location when viewing notifications

### Data Retention

- Notifications stored for 30 days
- Deleted notifications removed immediately
- Preferences saved indefinitely
