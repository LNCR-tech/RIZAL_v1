# User Manual

This guide covers the main Aura workflows for students, event managers, campus administrators, school IT, and platform administrators.

## Main features

### Students

- Check in to events.
- View attendance history.
- Receive notifications.
- Manage profile information.
- Use the mobile app.

### Event managers: SSG, SG, ORG

- Create events.
- Configure event schedules and geofence rules.
- Monitor attendance.
- Generate reports.
- Send event notifications.

### Administrators: platform admin, campus admin, school IT

- Configure school structure.
- Manage users and imports.
- Review system-wide reports.
- Manage technical configuration.
- Access protected technical documentation when authorized.
