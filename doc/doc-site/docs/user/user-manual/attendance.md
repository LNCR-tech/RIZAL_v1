# Attendance

Learn how to check in to events and view your attendance records.

## Checking In

### Face Recognition (Primary Method)
1. Approach the kiosk or use mobile app
2. Look at the camera
3. Wait for face verification
4. Confirmation message appears

### Manual Entry (Operator-Driven)
Operators (SSG, SG, ORG, Admin) can manually record attendance:
1. Select student from list
2. Confirm attendance
3. Record is created

## Viewing Your Records

1. Go to Dashboard
2. Click "My Attendance"
3. View your attendance history
4. Filter by date or event

## Troubleshooting

**Face not recognized?**
- Ensure good lighting
- Remove glasses or hats if needed
- Hold steady and look directly at camera
- Try again or ask operator for manual entry

**Location verification failed?**
- Ensure you're at the event location
- Enable precise location/GPS
- Wait for stronger GPS signal
- Check if you're within the event radius

**Already checked in?**
- You can only check in once per event
- Contact event organizer if there's an error
