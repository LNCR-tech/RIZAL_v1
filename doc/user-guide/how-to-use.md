## Login and Account Basics

1. Open Aura and sign in with your school account.
2. If prompted, complete first-login password change at `/change-password`.
3. Manage account security under `/profile/security`.
4. Complete first-time face enrollment at `/face-registration` when required.

---

## Student Daily Flow (`/dashboard`)

1. Open `Home` to see current notices and status.
2. Check `Schedule` for upcoming events.
3. Open event attendance pages for check-in/check-out details.
4. Review `Analytics` and attendance history.
5. Use `Aura Chat` (`/dashboard/chat`) when enabled.

---

## School IT / Campus Admin Flow (`/workspace`)

1. Manage users in `Users` and assign departments/programs.
2. Run bulk onboarding via `Import` (`/workspace/import`).
3. Configure student council and governance-related records as needed.
4. Manage schedules and monitor attendance.
5. Update school settings and review reports.

---

## Governance / SSG Officer Flow (`/governance`)

1. Create and manage events.
2. Open event details and configure attendance windows.
3. Run attendance during live events (manual or face scan).
4. Review governance reports and sanctions workflows when enabled.
5. Use `Aura Chat` (`/governance/chat`) when available.

---

## Platform/Admin Flow (`/admin`)

1. Review platform overview and school/account health.
2. Manage schools and accounts based on assigned permissions.
3. Access oversight and reports.
4. Use `Aura Chat` (`/admin/chat`) when enabled.

---

## Face Attendance Flow

1. Open the event attendance capture screen.
2. Ensure camera permission is granted.
3. Capture face scan with clear lighting and stable framing.
4. Wait for verification result.
5. If verification fails, perform officer-assisted manual attendance.

---

## Quick Troubleshooting

| Issue | Likely Cause | Action |
|---|---|---|
| Cannot log in | Wrong credentials or inactive account | Verify credentials and contact Campus Admin |
| Data not loading | Network issue or backend outage | Check connection, refresh, then retry |
| Face scan fails | Missing profile, poor light, camera permission denied | Re-register face, improve lighting, allow camera |
| Unexpected late status | Event timing windows are strict | Ask event officer/admin to review event window config |
